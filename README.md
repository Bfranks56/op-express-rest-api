# Express REST API Tutorial

A bare bones Node.js project for tutorial purposes.

## Getting Started

1. Install Node.js dependencies (if any):
   ```bash
   npm install
   ```

2. Run the application:
   ```bash
   npm start
   ```
   
   Or for development:
   ```bash
   npm run dev
   ```

## Project Structure
